# chip8
A multithreaded chip 8 emulator written in C++17 with SDL

![space-invaders](https://github.com/gregstula/chip8/screenshot.png)


## Compiling
Requires: SDL2

Compiling is easy with cmake.

Using cmake
`cmake -S . -B build`
`cmake --build build`

## How to use
`.build/chip8 [rom]`
